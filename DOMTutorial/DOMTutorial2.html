Tutorial 2: Manipulating the DOM

Тема 2: Манипулирование деревом DOM

This tutorial will teach you how to manipulate the DOM from JavaScript. Knowledge of how to access the DOM from JavaScript is assumed. There are five short lessons, followed by one exercise.

Эта тема посвящена вопросам манипулирования дерева DOM через JavaScript. Для изучения необходимы знания о получении доступа к дереву DOM через JavaScript. Здесь представлены пять коротких упражнений и одно практическое занятие.

It is highly recommended that you open your browser's JavaScript console now. When stuff doesn't work, there's usually an error printed in the console. The console is your friend.

Настоятельно рекомендуется открыть консоль JavaScript в браузере. Если что-то не работает, то ошибки обычно выводятся в консоль. Таким образом, консоль - ваш друг.

Lesson 1: innerHTML

Упражнение 1: innerHTML

The simplest way to to change the DOM is to assign some HTML to an element's innerHTML[?] property. This will erase the element's current content and replace it with your own.

Наиболее простой способ изменить DOM - это получить HTML элемента через свойство innerHTML[?]. Присваивая значения этому свойству приведет к замене текущего HTML содержимого элемента.

E.g. element.innerHTML = 'Hello<br>World'

Например: element.innerHTML = 'Здравствуй<br>Мир!'

Write some JavaScript code below that removes both the text and the star, and replaces them with a star that's 'on'. There is a 'star_on.gif' image on this server.

Напишите JavaScript код внизу для замены текста и изображения выключенной звезды этого span на изображение включенной звезды ('on'). Изображение 'star_on.gif' находится в этой директории.


 

Sample solution Решение document.getElementById('field').innerHTML = '<img src="star_on.gif">';

Lesson 2: removeChild

Упражнение 2: removeChild

In the previous lesson we used innerHTML. Never use innerHTML again. Although it is simple, fast and convenient, it opens a boat load of security holes, both obvious and subtle. Trust us. The rest of this tutorial will examine the more complicated, but safe alternative.

В предыдущем упражнении мы использовали свойство innerHTML. Больше никогда не используйте свойсвто innerHTML. Не смотря на то, что это просто, быстро и удобно, это может привести к проблемам с безопасностью, как явным так и скрытым. Доверяте нам. Остальную часть этого учебника мы посвятим более сложным, но безопастным альтернативам.

Since destroying something is easier than building something, let's start by destroying a node. Every node has a method removeChild[?] that takes as an argument the node that is to be destroyed. Note that this method needs to be called on the offending node's parent.

E.g. node1.removeChild(node2)

В связи с тем, что разрушать что-то проще, чем что-то строить, давайте начнем с удаления узла. Каждый узел имеет метод removeChild[?] который принимает в качестве аргумента узел которный должен быть удален. Отметим, что данный метод должен вызываться в родительском элементе, который содержит удаляемый узел

Пример: node1.removeChild(node2)
Здесь node1 - родительский элемент содержащий дочерний узел node2

Вот span который содержит некий текст и изображение звезды:   Shooting

Напишите JavaScript код внизу для удаления звезды.


 

Sample solution Решение var star = document.getElementById('fallenstar');
star.parentNode.removeChild(star);

Lesson 3: appendChild

Упражнение 3: appendChild

Nodes may be added as children of another node with the appendChild[?] method. This method is called on the new parent and takes the child as its argument. If the child you are appending is an existing element in the DOM, it will be automatically removed from its existing location. Any grandchildren will follow the child.

E.g. node1.appendChild(node2)

Узлы могут быть добавлены как дочерние к другим узлам через метод appendChild[?]. Этот метод вызывается в родительском узле и принимает в качестве аргумента дочерний узел. Если добавляемый дочерний узел существует в дереве DOM, то он будет автоматически удален с его текущего места (т.е. происходит перемещение узла). Все вложенные дочерние узлы также будут перемещены.

Пример: node1.appendChild(node2)

Write some JavaScript code below which moves the star from 'source' to the end of 'target', without changing the text.

Напишите JavaScript код внизу для перемещения звезды из 'source' в конец 'target' без изменения текста.


 

Sample solution Решение document.getElementById('target').appendChild(document.getElementById('source').lastChild);
