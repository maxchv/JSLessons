Tutorial 1: Accessing the DOM

Тема 1: Работа с Объектной моделью документа (Document Object Model)

This tutorial will teach you how to access the DOM from JavaScript. Basic knowledge of JavaScript and HTML is assumed. There are five short lessons, followed by one exercise.

В этом учебнике вы познакомитесь с получением доступа к Объектной модели документа (DOM) через JavaScript. Предполагается, что обучающиеся имеют базовые знания JavaScript и HTML. Здесь представлены пять коротких упражнения и одно практическое задание.

It is highly recommended that you open your browser's JavaScript console now. When stuff doesn't work, there's usually an error printed in the console. The console is your friend.

Настоятельно рекомендуется открыть консоль JavaScript в браузере. Если что-то не работает, то ошибки обычно выводятся в консоль. Таким образом, консоль - ваш друг.

Lesson 1: getElementById

Упражнение 1: Получение элемента по его идентификатору через getElementById

Finding a node on a page is the most fundamental task when working with the DOM from JavaScript. The most useful function is document.getElementById('elementId')[?]. This function takes a string and returns the DOM element with that ID.

Поиск узла на странице это одна из основных задач при работе с DOM из JavaScript. Наиболее полезная функция document.getElementById('elementId')[?]. Эта функция получает строку и возвращает DOM элемент с указанным идентификатором 'elementId'.

Here is an image of a star:Это картинка звездочки:  

This is the DOM tree for the above image:

Это дерево DOM для указанной картинки:

<img id="star" src="star_off.gif">

Write some JavaScript code below that finds the above image element using getElementById, then sets the element's src[?] property so that the star becomes on. There is a 'star_on.gif' image on this server.

Напишите JavaScript код для поиска элемента картинка используя getElementById, потом установите для этого элемента атрибут src[?] так, что бы звезда была включена. На сервере есть картинка с именем 'star_on.gif'.


 

Sample solution Решение document.getElementById('star').src = 'star_on.gif';

Lesson 2: childNodes

Упражнение 2: поиск дочерних элементов используя childNodes

Sometimes the node you want to manipulate does not have an ID. This is particularly true of text nodes, which can't have IDs. But there may still be ways to find a node that doesn't have an ID. Each node has a childNodes[?] property that contains an ordered array of all its children. One can index into this array.

Иногда узлы, которыми необходимо манипулировать, не имеют идентификаторов. Это, в частности, справедливо для текстовых узлов, которые не могут иметь идентификаторов. Но все же существует способ найти узел для которого не установлен атрибут id. Каждый узел имеет childNodes[?] свойство которое содержит упорядоченный массив всех дочерних элементов Таким образом можно перебрать индексы этого массива.

Here is a span containing three stars: Вот span содержащий три звезды:  

This is the DOM tree for the above span:

Дерево DOM для указанного выше span:

<span id="stars">
┣ <img src="star_off.gif">
┣ <img src="star_off.gif">
┗ <img src="star_off.gif">
</span>

Write some JavaScript code below that finds the above span element using getElementById, finds the middle star using childNodes, then sets the element's src property so that the star becomes 'on'. The other two stars should be 'off'. Remember that arrays in JavaScript start with index 0.

Напишите JavaScript код внизу для поиска элемента span используя getElementById, затем найти среднюю звезду используя childNodes, затем установить свойство src для этого элемента что бы звезда была включена ('on'). Остальные две звезды должны быть выключены ('off'). Помните, что массивы в JavaScript начинаються с индекса 0.


 

Sample solution Решение document.getElementById('stars').childNodes[1].src = 'star_on.gif';

Lesson 3: firstChild & lastChild

Упражнение 3: поиск дочерних элементов через firstChild & lastChild

There's a shortcut for finding the first and last child nodes, Существует сокращение для поиска первого и последнего дочернего узла, firstChild[?] and и lastChild[?].

Here is a phrase with a star: Вот фраза со звездой:   Twinkle twinkle little star.

This is the DOM tree for the above span:

Это дерево DOM для указанного выше span:

<span id="phrase">
┣ Twinkle twinkle little
┗ <u>
  ┣ <img src="star_off.gif">
  ┗ star
  </u>
</span>

Write some JavaScript code below that finds the above span element using getElementById, finds the star using firstChild and/or lastChild, then sets the element's src property so that the star becomes 'on'.

Напишите JavaScript код внизу который отыскивает span элемент используя getElementById, находит звезду используя firstChild и/или lastChild, затем устанавливает свойство src элемента так, что бы звезда была включена ('on').


 

Sample solution Решение document.getElementById('phrase').lastChild.firstChild.src = 'star_on.gif';

Lesson 4: previousSibling & nextSibling

Упражнение 4: поиск соседей используя previousSibling & nextSibling

Nodes also have Узлы также имеют previousSibling[?] and и nextSibling[?] properties that allow one to slide sideways on the DOM tree. свойства, которые позволяют перемещаться в по дереву DOM на этом уровне.

Here is another span containing three stars: Вот другой span содержащий три звезды:  

This is the DOM tree for the above span:

Это дерево DOM для указанного выше span:

<span>
┣ <img src="star_off.gif">
┣ <img src="star_off.gif">
┗ <img id="laststar" src="star_off.gif">
</span>

Write some JavaScript code below that finds the last star, then walks back to turn the first star 'on'. The other two stars should be 'off'.

Напишите JavaScript код внизу который позволяет найти последнюю звезду, потом двигается назад для включения ('on') первой звезды. Остальные две звезды должны быть выключены ('off').


 

Sample solution Решение document.getElementById('laststar').previousSibling.previousSibling.src = 'star_on.gif';

Lesson 5: parentNode

Упражнение 5: поиск родительских элементов через parentNode

The last DOM-walking property is Последнее свойство для поиска DOM элементов parentNode[?]. As might be expected, this returns the node that encloses the current node. Как можно ожидать, он возвращает узел содержащий текущий узел.

Here is a more complex snippet: Вот более сложный пример:   Twinkle twinkle little

This is the DOM tree for the above span:

Это дерево DOM для указанного выше span:

<span>
┣ <b>
┃ ┗ <small id="start_here">
┃   ┗ Twinkle
┃   </small>
┃ </b>
┗ <i>
  ┣ twinkle little
  ┗ <img src="star_off.gif">
  </i>
<span>

Write some JavaScript code below that starts with the small tag, then walks to the star and turns it 'on'.
Bonus: Not counting getElementById and src, can you do it in only three steps?

Напишить JavaScript код внизу что бы начать с элемента с тегом small, затем перейти к звезде и включите ее ('on').
Бонус: Не считая применения метода getElementById и src, можете сделать это только в три шага?


 

Sample solution Решение document.getElementById('start_here').parentNode.nextSibling.lastChild.src = 'star_on.gif';

Exercise: Tree Walker

Практическое занятие: Tree Walker

One final detail is that the firstChild, lastChild, previousSibling, nextSibling, and parentNode properties return null if there is no such child, sibling or parent node. Likewise, the childNodes array is of length zero if there are no child nodes.

Последнее замечание - свойства firstChild, lastChild, previousSibling, nextSibling, и parentNode возвращают null если не найден дочерний, соседний или родительский узел. Аналогично массив childNodes имеет нулевую длину (свойство length равно нулю) при отсутствии дочерних узлов.

You now know the API for navigating your way around any DOM tree. For this exercise, put all these pieces together to write a function called nextNode that can be used to walk through all the nodes of a DOM tree. The function should take a node as an argument, and return the next node in the tree. For example, the following loop would visit every node once, and only once:

Теперь вы знаете API для навигации по любому дереву DOM. Для этого практического занятия соберите все эти кусочки знаний вместе для написания функции nextNode, которая может быть использована для перемещения по всем узлам дерева DOM. Функция должна получать в качестве аргумент узел и возвращать следующий узел в дереве. Например, следующий цикл будет проходить по каждому узлу лишь один раз: 

  var node = document.body;
  do {
    node = nextNode(node);
  } while (node != null)

This is not an easy exercise, it requires careful thought. Beware of infinite loops which may crash your browser.

Это не простое практическое задание, оно требует тщательной проработки. Остерегайтесь бесконечных циклов, которые могут привести к краху вашего обозревателя.


Test your nextNode function by repeatedly clicking the "Execute single step" button. It will highlight each node in sequence, starting with the title of this exercise. Elements (which correspond with HTML tags) receive a green border, text nodes receive a green background.

Проверте вашу функцию nextNode повторно нажимая на кнопку "Выполнить единственный шаг". При правильном решении будет подсвечиваться каждый узел в дереве, начиная с заголовка этого практического задания. Элементы (которые соответствуют HTML тегам) будут иметь зеленую границу, текстовые узлы будут иметь зеленый фон.
